# Projeto-redução-de-custos-numa-empresa-farmacêutica
O objetivo do projeto é propor a utilização de 3 serviços da AWS para diminuir os custos de uma empresa farmacêutica. 

  RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

  Data: 12 de Janeiro de 2026 
  
  Empresa: Abstergo Industries
  
  Responsável: Lázaro Leonardo Peña Reyes 

  Introdução

  Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por o especialista Lázaro. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos e otimizar as operações de distribuição de medicamentos.

  Descrição do Projeto

  O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

  Etapa 1: Amazon S3 (Simple Storage Service)

  • Nome da ferramenta: Amazon S3

  • Foco da ferramenta: Armazenamento de objetos escalável, seguro e de baixo custo.

  • Descrição de caso de uso: O Amazon S3 será utilizado para armazenar de forma segura e econômica todos os documentos da empresa, como contratos com fornecedores e farmácias, faturas, relatórios financeiros, dados de inventário, registros de pedidos e dados históricos de vendas. A migração desses dados para o S3 elimina a necessidade de servidores de arquivos locais caros e complexos de manter, reduzindo custos de infraestrutura e manutenção. Além disso, oferece alta durabilidade e disponibilidade dos dados, essenciais para a continuidade dos negócios.

  Etapa 2: AWS Lambda

  • Nome da ferramenta: AWS Lambda

  • Foco da ferramenta: Computação sem servidor (serverless) para execução de código orientada a eventos.

  • Descrição de caso de uso: O AWS Lambda será empregado para automatizar processos de backend críticos na cadeia de distribuição. Isso inclui o processamento automático de novos pedidos recebidos de farmácias, a atualização automática dos níveis de estoque quando um medicamento é distribuído ou recebido, e a geração programada de relatórios diários ou semanais sobre o fluxo de distribuição e vendas. Ao utilizar o Lambda, a empresa paga apenas pelo tempo de computação consumido, eliminando custos com provisionamento e gerenciamento de servidores, o que resulta em uma significativa redução de custos operacionais e maior agilidade.

  Etapa 3: Amazon RDS (Relational Database Service)

  • Nome da ferramenta: Amazon RDS (com PostgreSQL ou MySQL)

  • Foco da ferramenta: Serviço de banco de dados relacional gerenciado e escalável.

  • Descrição de caso de uso: O Amazon RDS será a base para o banco de dados central da Abstergo Industries, hospedando informações cruciais como catálogo de produtos farmacêuticos, dados detalhados de clientes e farmácias parceiras, histórico completo de pedidos e um sistema robusto de gerenciamento de inventário em tempo real. A utilização do RDS elimina a complexidade e o custo da administração de bancos de dados on-premise, incluindo tarefas como backups, patches e escalabilidade. Isso permite que a equipe se concentre nas operações de negócios, enquanto o AWS gerencia a infraestrutura do banco de dados de forma confiável e econômica.

  Conclusão

  A implementação dos serviços AWS – Amazon S3, AWS Lambda e Amazon RDS – na empresa Abstergo Industries tem como esperado a redução significativa de custos operacionais e de infraestrutura, a otimização da gestão de dados e processos logísticos, e o aumento da eficiência e produtividade da empresa. A transição para a nuvem permitirá à Abstergo Industries focar em seu core business de distribuição de medicamentos, com uma infraestrutura de TI mais ágil, escalável e econômica. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
